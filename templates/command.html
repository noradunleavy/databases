Mastering the Command Line

SET UP:

1. Open the terminal (MAC OS & Linux)

LAB ONE: AROUND THE WORLD

1. Use the command line to make it to the "around-the-world" Directory! See hints below on what commands to use!

2. Use the command line to make sure that all of the cities are in the right country and each city folder has the right monuments. You wouldn't want to end up in Greece looking for the Eiffel Tower!

3. Add new cities and monuments that YOU would like to visit around the world using the command line :)))).

EXTENSION: Mix up the cities, and go into the city of your choice... stay in that city and do what is listed above again !!!

Help someone if you finish all of this!

Helpful List of commands that might be needed!

-ls (list files that are in the current directory (also known as a folder in the englist language!))

-pwd (print current directory that you are in!)

-cd (change directory)

- . (the current directory)

- .. (parent directory (also known as the folder above the one you are in!))

-touch (create file)

-mkdir (make a folder)

-mv ____ _____ (move ____ to ____)

-cp _____ _____ (copy _____ to ____)

-gedit [file] (open the file and edit the text!)
