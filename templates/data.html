Implementing a Database

TASK:

1. While in Atom, go to "File", "Open", and find the entire folder zip file you downloaded "app-engine-starter"

2. Take a look through the main.py file. This is where we accept some of the user inputs for our database (like your name!), and is sometimes called the controller!

There are three main components: the imports (line webapp2), the handlers (these respond to our requests!), and the app configuration.

3. To launch the app in the command line, copy over the following: dev_appserver.py app.yaml

4. The root route (app = webapp2.WSGIApplication([('/', MainPage), #the root route], debug=True)) is the location where the handler is directed to.

You know how if you want to go to a specific part of a web application, you use /???? where ???? specifies one part of the application! This is where we use it comes from.

5. TEMPLATES!!! WHAT WOULD WE DO WITHOUT THEM? these gems allow us to use external data,

but you have to import them to put them to use as part of Google App Engine...my personal favorite "import webapp2" :)

6. GETTING USER DATA AHHHHHHHH -> we use a form :)

7. To get the data you asked for.... use self.request.get() with or without an argument (which is what does inside the () of a method!)

8. Want to deploy? Create an account on the Google Platform Developer's console

and run in the command line the following: gcloud app deploy app.yaml --project [STUDENTS_PROJECT_ID]

.... and go to http://[STUDENTS__PROJECT_ID].appspot.com to see your innovation!
